บทเรียนพาคุณสร้างของจริง ทุกบทเรียนในที่นี้:

- เริ่มจาก `cephalon new` และจบที่แอปที่รันได้ มี observability และดีพลอยได้
- แสดง diff เต็มของแต่ละขั้น ไม่ใช่แค่ snippet ของ happy path
- อธิบาย **เหตุผล** ของการตัดสินใจ ไม่ใช่แค่ **วิธี**
- จบด้วยเช็คลิสต์ว่าจะทำอะไรต่อหลังจากบทเรียนนี้

ถ้าเพิ่งเริ่มใช้ CephalonEngine ให้เริ่มที่ **[สร้างแอปแรก](/th/0-1-0-preview/tutorials/first-app/)**

## ข้อตกลงในบทเรียน

| ข้อตกลง | ความหมาย |
| --- | --- |
| **หัวข้อต่อขั้น** | แต่ละขั้นรันได้อิสระ หยุดที่ขั้นไหนก็ได้ |
| **diff เต็ม** | เราแสดงไฟล์ก่อนและหลังการเปลี่ยนแปลง |
| **"ทำไมถึงได้ผล"** | คำอธิบายเชิงสถาปัตยกรรมของแต่ละการเปลี่ยนแปลง |
| **"จุดที่พลาดได้"** | ความผิดพลาดที่เราเจอ เพื่อให้คุณไม่ต้องเจอซ้ำ |
| **"ทำต่ออะไร"** | ลิงก์ไปยังหน้า Guide / Reference ที่เกี่ยวข้อง |

> หมายเหตุ: หน้าบทเรียนส่วนใหญ่ในภาษาไทยกำลังทยอยแปล หน้าใดที่ยังไม่ได้แปลจะแสดงเป็นภาษาอังกฤษอัตโนมัติ — เนื้อหายังคงครบถ้วน
