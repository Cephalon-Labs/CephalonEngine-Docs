### Classes

 [AgentToolExecutionOutcomes](/0-1-0-preview/reference/api/cephalon-abstractions-agentics-agenttoolexecutionoutcomes/)

Defines stable outcome identifiers for agent-tool execution observations.

 [AgentToolExecutionRequest](/0-1-0-preview/reference/api/cephalon-abstractions-agentics-agenttoolexecutionrequest/)

Describes one request to execute an agent tool through the Cephalon-managed agentics runtime.

 [AgentToolExecutionResult](/0-1-0-preview/reference/api/cephalon-abstractions-agentics-agenttoolexecutionresult/)

Describes the result returned by one managed agent-tool executor.

 [AgentToolRunState](/0-1-0-preview/reference/api/cephalon-abstractions-agentics-agenttoolrunstate/)

Describes the latest operator-facing runtime state reported for one agent-tool run.

 [IAgentToolDispatcher](/0-1-0-preview/reference/api/cephalon-abstractions-agentics-iagenttooldispatcher/)

 [IAgentToolRunCatalog](/0-1-0-preview/reference/api/cephalon-abstractions-agentics-iagenttoolruncatalog/)

Exposes the latest reported runtime state for agent-tool runs.
