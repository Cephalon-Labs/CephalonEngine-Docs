`Cephalon.Edge` is the companion package for edge-native delivery runtime services. It pairs with Traefik or the Kubernetes Gateway API to ship hot-path routing and signed module delivery to the edge.

## What you'll build

- a Cephalon host fronted by Traefik or KGW.
- hot-path routes that bypass the full pipeline for cached responses.
- signed module delivery so edge nodes refuse unverified modules.
