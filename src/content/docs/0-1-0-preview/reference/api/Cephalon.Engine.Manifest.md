### Classes

 [CapabilityManifest](/0-1-0-preview/reference/api/cephalon-engine-manifest-capabilitymanifest/)

Describes a capability exposed by a module in the runtime manifest.

 [ModuleManifest](/0-1-0-preview/reference/api/cephalon-engine-manifest-modulemanifest/)

Describes a single module that participates in the built runtime.

 [PackageDependencyManifest](/0-1-0-preview/reference/api/cephalon-engine-manifest-packagedependencymanifest/)

Describes a package-to-package dependency declared by a loaded Cephalon package.

 [PackageDistributionManifest](/0-1-0-preview/reference/api/cephalon-engine-manifest-packagedistributionmanifest/)

Describes where an independently shipped package is distributed from.

 [PackageManifest](/0-1-0-preview/reference/api/cephalon-engine-manifest-packagemanifest/)

Describes a package that contributed one or more modules to the built runtime.

 [PackageProvenanceManifest](/0-1-0-preview/reference/api/cephalon-engine-manifest-packageprovenancemanifest/)

Describes provenance metadata declared for an independently shipped package.

 [PackageSignatureManifest](/0-1-0-preview/reference/api/cephalon-engine-manifest-packagesignaturemanifest/)

Describes a single package signature declared by a package manifest and the outcome of verifying it.

 [RuntimeManifest](/0-1-0-preview/reference/api/cephalon-engine-manifest-runtimemanifest/)

Represents the immutable manifest produced when a Cephalon runtime is built.
