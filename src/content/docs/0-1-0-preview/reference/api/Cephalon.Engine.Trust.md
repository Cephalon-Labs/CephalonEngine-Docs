### Classes

 [CapabilityPolicyDecision](/0-1-0-preview/reference/api/cephalon-engine-trust-capabilitypolicydecision/)

Describes the evaluated trust decision for a single capability.

 [CapabilityPolicyEvaluator](/0-1-0-preview/reference/api/cephalon-engine-trust-capabilitypolicyevaluator/)

Evaluates capability and package trust decisions against the current trust policy snapshot.

 [PackageSignatureTrustDecision](/0-1-0-preview/reference/api/cephalon-engine-trust-packagesignaturetrustdecision/)

Describes the trust and verification outcome for a single package signature.

 [PackageTrustDecision](/0-1-0-preview/reference/api/cephalon-engine-trust-packagetrustdecision/)

Describes the trust outcome for a package after package metadata, signature verification, and host trust rules have been evaluated.

 [TrustSnapshot](/0-1-0-preview/reference/api/cephalon-engine-trust-trustsnapshot/)

Captures the effective trust policy together with evaluated package and capability decisions.
