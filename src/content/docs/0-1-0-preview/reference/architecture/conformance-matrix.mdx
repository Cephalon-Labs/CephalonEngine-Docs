The conformance matrix is the canonical cross-reference between every shipped package and the public surfaces it participates in.

## What each column means

| Column | Meaning |
| --- | --- |
| **Package** | The NuGet package name. |
| **Maturity** | Current M0–M4 label. |
| **Engine contract** | Routes / catalogs / contributors the package implements. |
| **Manifest contribution** | What the package writes into the runtime manifest. |
| **Telemetry attributes** | Per-package resource attributes contributed to telemetry. |
| **Configuration prefix** | The `Engine:<prefix>:*` section the package owns. |
