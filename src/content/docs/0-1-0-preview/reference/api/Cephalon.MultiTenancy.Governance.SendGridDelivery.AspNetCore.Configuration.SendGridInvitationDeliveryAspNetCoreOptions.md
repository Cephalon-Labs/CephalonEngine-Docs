```csharp
public bool NormalizeProviderMessageIdFromSgMessageId { get; set; }
```

#### Property Value

 [bool](https://learn.microsoft.com/dotnet/api/system.boolean)

#### Remarks

Twilio SendGrid recommends storing the Mail Send API <code>X-Message-ID</code> response header to correlate Event
Webhook posts. Event Webhook payloads carry <code>sg_message_id</code>; the prefix commonly matches the stored
<code>X-Message-ID</code>, so this option keeps Cephalon's provider-message guard usable without weakening it.


#### Property Value
