### Classes

 [EdgeNodeDescriptor](/0-1-0-preview/reference/api/cephalon-edge-services-edgenodedescriptor/)

Describes an edge node that can be surfaced through the edge runtime pack.

### Interfaces

 [IEdgeNodeCatalog](/0-1-0-preview/reference/api/cephalon-edge-services-iedgenodecatalog/)

Exposes the merged set of edge nodes available to the active edge runtime.

 [IEdgeNodeContributor](/0-1-0-preview/reference/api/cephalon-edge-services-iedgenodecontributor/)

Allows a module to contribute edge nodes into the active edge runtime pack.

 [IEdgeNodeRegistry](/0-1-0-preview/reference/api/cephalon-edge-services-iedgenoderegistry/)

Collects edge node descriptors contributed to the active edge runtime pack.
