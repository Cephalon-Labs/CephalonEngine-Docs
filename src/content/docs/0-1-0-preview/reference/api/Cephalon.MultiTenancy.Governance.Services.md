 [ITenantMembershipEvaluator](/0-1-0-preview/reference/api/cephalon-multitenancy-governance-services-itenantmembershipevaluator/)

Evaluates whether a principal has active membership in a tenant.

 [ITenantMembershipRegistry](/0-1-0-preview/reference/api/cephalon-multitenancy-governance-services-itenantmembershipregistry/)

Collects tenant memberships contributed to the active governance runtime.

 [ITenantMembershipStore](/0-1-0-preview/reference/api/cephalon-multitenancy-governance-services-itenantmembershipstore/)

Stores runtime tenant memberships managed by the multi-tenancy governance companion pack.
