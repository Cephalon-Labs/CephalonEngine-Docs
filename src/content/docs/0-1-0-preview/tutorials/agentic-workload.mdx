`Cephalon.Agentics` provides runtime services for agentic workloads — a typed task graph, a tool registry, scheduled agents, and agent-to-event handoff into the broader Cephalon eventing capability.

## What you'll build

- a typed task graph with retries, branches, and per-node telemetry.
- a tool registry that exposes existing modules' behaviors to agents.
- a scheduled agent that wakes daily and publishes an event when complete.
